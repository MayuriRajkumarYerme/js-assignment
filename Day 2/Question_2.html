Question 2

Explain with examples the remaining methods for string and array

1]String

1)substring() : Extracts the characters from a string, between two specified indices.
Syntex - string.substring(start, end)

2)replace() : Searches a string for a specified value, or a regular expression, and returns a new string where the specified values are replaced
Syntex - string.replace(searchvalue, newvalue)

3)slice() : Extracts a part of a string and returns a new string
Syntex - string.slice(start, end)

2]Array

1) copyWithin() : Copies array elements within the array, to and from specified positions.
Syntex - array.copyWithin(target, start, end)

2) fill() : Fill the elements in an array with a static value.
Syntex - array.fill(value, start, end)

3)forEach() : Calls a function for each array element.
Syntex - array.forEach(function(currentValue, index, arr), thisValue)
